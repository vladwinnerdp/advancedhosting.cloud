== Description ==

The main task of this plugin is a send your `Contact Form 7` forms directly to your SendPulse account.

= Features =

* Integrate your `Contact Form 7` forms with SendPulse;
* Integrate unlimited `Contact Form 7` forms;
* When submitting a form, the plugin creates a contact in the selected Mailing list;
* Mailing lists are loaded from the SendPulse (including variables);
* You can set up each form personally, specify which information your want to get;
* Supports getting `utm` params from the `URL`;
* Supports for sending `GA Client ID`;
* Supports for sending `roistat_visit`;
* Multiple `Mailing lists` support;
* Double Opt-In support;
* Image previews;
* Super easy to set-up;

== Installation ==

1. Extract `contact-form-7-sendpulse-integration.zip` and upload it to your `WordPress` plugin directory
(usually /wp-content/plugins ), or upload the zip file directly from the WordPress plugins page.
Once completed, visit your plugins page.
2. Be sure `Contact Form 7` Plugin is enabled.
3. Activate the plugin through the `Plugins` menu in WordPress.
4. Go to the `Contact Form 7` -> `Integration`.
5. Find `Integration with `SendPulse` and click the button `Go to setup`.
6. Insert in the field `API ID` and `API Secret`. Your personal API ID and API Secret can be found - Account Settings > API.
7. Save settings.
8. When editing forms your can see the tab `SendPulse`.
