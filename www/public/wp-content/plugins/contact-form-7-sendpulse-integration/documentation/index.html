A) About - top

The main task of this plugin is a send your Contact Form 7 forms directly to your SendPulse account.

* Integrate your `Contact Form 7` forms with SendPulse;
* Integrate unlimited `Contact Form 7` forms;
* When submitting a form, the plugin creates a contact in the selected Mailing list;
* Mailing lists are loaded from the SendPulse (including variables);
* You can set up each form personally, specify which information your want to get;
* Supports getting `utm` params from the `URL`;
* Supports for sending `GA Client ID`;
* Supports for sending `roistat_visit`;
* Multiple `Mailing lists` support;
* Double Opt-In support;
* Image previews;
* Super easy to set-up;

B) Step 1 - Integration - top

Go to the Contact Form 7 -> Integration.

Find "Integration with SendPulse" and click the button "Go to setup".

Insert in the field `API ID` and `API Secret`. Your personal API ID and API Secret can be found - Account Settings > API.

  1. Login to sendpulse.com
  2. In the top right corner, click on the dropdown menu with the circle avatar.
  3. Choose `Account Settings`.
  4. Underneath the 'Account Settings' heading, choose API from the list of tabs.
  5. If it is not already activated, click Activate REST API
  6. You will see your ID and Secret keys.

Save settings.

C) Step 2 - Form settings - top

Go to the Contact Form 7. Open the form which your want to send to SendPulse.

Open tab SendPulse.

Set checkbox "Enable send".

Fill fields and click "Save" button.

When sending the form, the data will be transferred to SendPulse.
